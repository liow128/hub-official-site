# hub-official-site

本仓库用于归档和发布多个独立的 HTML 页面。这些页面作为项目相关的静态资源，可供浏览、测试或引用。

## 项目简介

这是一个静态页面归档仓库，主要用途包括：

- 存放历史版本或特定用途的 HTML 文件
- 提供可直接访问的页面副本
- 作为项目相关静态内容的集中存放地

仓库本身不针对任何特定域名或网站，仅用于归档和发布独立的 HTML 内容。

## 目录结构

```
.
├── index.html          # 页面索引（如有）
├── pages/              # 归档页面存放目录
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   └── ...
└── README.md           # 本文件
```

- `pages/`：存放各个独立 HTML 页面文件
- `assets/`：存放页面所需的公共静态资源

## 页面归档说明

每个 HTML 页面均为独立完整的文件，可直接在浏览器中打开。页面之间相互独立，不依赖外部服务或特定网络环境。

归档页面可能包含：

- 项目说明或展示页面
- 工具或功能演示页面
- 其他与项目相关的静态内容

页面命名尽量反映其内容或用途，便于检索。

## 维护说明

- 添加新页面时，请确保文件完整自包含，或将依赖资源放入 `assets/` 目录
- 建议在提交信息中简要说明新增或修改的页面用途
- 如需删除或更新页面，请保持仓库目录结构清晰
